# 🐦 Phoenix AI Chatbot

🌐 **Live Demo:** [Visit the site](https://my-chat-phoenix.vercel.app/)

## 🌟 Overview
An AI Chatbot website built with **React**, **TailwindCSS**, and **Appwrite**, deployed on **Vercel**.

- 💬 Real‑time AI Chat
- ⚡️ Modern, Responsive UI
- 🔐 Appwrite Backend
- 🌐 Deployed on Vercel

## 🛠️ Technologies
- React
- TailwindCSS
- Appwrite
- Vercel
